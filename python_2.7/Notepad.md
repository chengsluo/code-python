# spider学习笔记
## urllib
非常常用的python网络库，功能广泛

## cookielib
提供了可存储和操作cookie的对象与方法, 以便于与urllib模块配合使用来访问Internet

## httplib
相对底层的HTTP请求处理模块, 不如urllib模块封装的层次高,但能更加灵活地对通信进行控制

## beautifulSoup
一个HTML/XML的解析器,可以很好地处理不规范标记并生成剖析树(parse tree)

## Scrapy
一个快速、高层的屏幕抓取和web抓取框架，用于抓取web站点并从页面中提取结构化数据。

## json
轻量级数据交换格式

## BeJson
同上

## rsa
密码模块，可以实现模拟登录
